# Executando:
<p>- Importante:
- Datasets de imagens se comportam melhor na escala de cinza<br>
- Tente não usar imagens grandes.<br>
- Quanto mais imagens em seu dataset melhor sera o resultado de seu Trainer.<br>
- Feche todo programa que consuma GPU durante a crição dos Datasets e Trainers, afinal muitas imagens estão sendo processadas.<br>
- Feche todos programas pois seu processador vai suar.<br>
- Sim este sistema roda sem GPU porém é muitissimo mais lento.<br> 
- Tente manter no minimo 100 imagens na pasta de datasets mundando o script de acordo com tutorial em video.<br>
- Quanto mais imagens inseridas nestas pastas melhor.<br>   
  </p><br>
